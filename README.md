# AI Study Engine

AI-powered study management platform.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with `GROQ_API_KEY=your_key`
3. Run: `streamlit run app.py`

## Features

- Flashcard Generation
- Quiz Generation
- Notes Summarization
- Answer Evaluation
- Matching Games
- Study Guide Builder
